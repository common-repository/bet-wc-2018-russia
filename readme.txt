Bet matches for WC 2018 Russia
You can use this plugin to bet matches on World Cup 2018 Russia with Yours friends.

= Shortcodes = 
U can use shortcodes to display in every place You want
[fms2018_mecze] - show bets
[fms2018_ranking] - show ranking 
[fms2018_stats] - show statistics

= Wtyczka dostępna w Języku Polskim = 
Typowanie Meczy na MŚ 2018 Rosja
Możesz użyć tej wtyczki, aby obstawiać mecze na Mistrzostwa Świata 2018 w Rosji ze swoimi znajomymi.

= Shortcodes = 
Możesz używać Shortcodes do wyświetlania w dowolnym miejscu.
